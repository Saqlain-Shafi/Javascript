JS OBJECT

JS objects are also like variables but they store named valus while as variables store only values.

A JS object is defined as object name and then followed by its property names and their values.

Example: fruit = {Name:"apple",Color:"red",variety:"Delicious";}

In the above example fruit is object name ,'name,color,etc' are property names and 'apple,red,etc' are property values. The properties are separated by using a comma.

JS PROPERTY

The JS property is assessed by two methods:

  1. objectname.propertyname
  2. objectname["propertyname"]
